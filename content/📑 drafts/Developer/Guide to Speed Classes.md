The SD Association came up with a way to standardize the speed ratings for memory cards. They are known as Speed Classes and refers to the absolute minimum sustained write speed for both SD and [microSD](https://www.kingston.com/en/blog/personal-storage/microsd-sd-memory-card-guide) cards. There are three types of speed class:

- Speed Class
- UHS Speed Class
- Video Speed Class
